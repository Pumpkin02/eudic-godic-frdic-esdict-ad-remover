修复欧路助手系列网页版（德法西语）全屏广告拦截问题的脚本。

# 欧路助手系列 - 全屏广告遮罩修复

本脚本用于解决欧路词典、德语助手、法语助手、西语助手网页版出现的“Google 全屏广告导致页面无法点击”的问题。

### 🚀 安装方式

1. 请先安装浏览器插件 [Tampermonkey (油猴)](https://www.tampermonkey.net/)。
2. **[点击此处一键安装本脚本](https://github.com/Pumpkin02/eudic-godic-frdic-esdict-ad-remover/raw/refs/heads/main/eudic-godic-frdic-esdict-ad-remover.user.js)**。

### 🛠 功能
- 自动移除全屏遮罩层。
- 恢复被广告锁定的页面滚动。
- 解决无法点击单词或输入框的问题。
